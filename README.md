# A Space Shooter Game
This is a space shooter game i made in pico 8 game engine from scratch.

# How to play it, if u want to??!?!!!?!?
- Arrow keys for left and right movement
- 'Z' for shooting
- shoot the enemies
- 'X' for restarting

# How to start the damn game tho!?!?!?!
Download the file spaceshooter_sfx.p8 and open it in pico 8.
